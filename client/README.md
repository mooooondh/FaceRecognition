# Demo Web-Page

## How to start?

Node.js가 설치 되어있어야 하고, React module 설치를 해야합니다.

```
git clone https://github.com/mooooondh/FaceRecognition.git
cd client
npm install
npm start
```

프로젝트를 클론하고, 리액트 프로젝트인 client 폴더에 들어갑니다.
client 폴더내에서 모듈을 받기위해 npm install 명령어를 사용하고 npm start로 시작합니다.
